# Web Scraping Project - Books Dataset

## Project Overview
This project demonstrates web scraping using Python, BeautifulSoup, and Pandas.

## Objective
Extract book information from Books to Scrape and store it in an Excel dataset.

## Data Collected
- Book Title
- Price
- Rating

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas
- Google Colab

## Output
- 1000 book records collected
- Dataset saved as books_dataset.xlsx
